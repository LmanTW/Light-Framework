# Installation
You can install Light-Framework in several ways, but we suggest you to just import it from `framework.light.tw`.

## Import
```js
import Light from '<url>'
```
* `framework.light.tw` | https://framework.light.tw/Asset/Light.mjs
* jsDelivr | https://cdn.jsdelivr.net/gh/LmanTW/Light-Framework/Assets/API.mjs (May not be the newest version)
