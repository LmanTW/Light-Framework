# What Is Light-Framework
Light-Framework is a really simple framework for building [SPA (Single Page Application)](https://en.wikipedia.org/wiki/Single-page_application). It provides APIs to both [JavaScript](https://en.wikipedia.org/wiki/JavaScript) and [HTML](https://en.wikipedia.org/wiki/HTML), and also allow developers to easily build plugins to extend the framework.

## Keep things simple
We want to keep things simple, so Light-Framework only contains some basic functionality, but we believe that's enough for building a [SPA (Single Page Application)](https://en.wikipedia.org/wiki/Single-page_application). Light-Framework is a really small framework (around 15kb), and we want to keep it this way. If you want any extra functionality, you can either find or build a plugin.
