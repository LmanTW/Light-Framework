Installation

You can install Light-Framework in several ways, but we suggest you to just import it from framework.light.tw.

ESM

// Import From [framework.light.tw]
import Light from 'https://framework.light.tw/Asset/Light.mjs'

// Import From jsDelivr (May not be the newest version)
import Light from 'https://cdn.jsdelivr.net/gh/LmanTW/Light-Framework/Assets/Light.mjs'

UMD

<script type="module" src="https://framework.light.tw/Asset/Light.global.mjs"></script>

<script type="module" src="https://cdn.jsdelivr.net/gh/LmanTW/Light-Framework/Assets/Light.global.mjs"></script>
