# Simple cache
Provides functionalities for caching data

## Installation
```js
// Import From [framework.light.tw]
import SimpleCache from 'https://framework.light.tw/Asset/Plugin/SimpleCache.mjs'

// Import From jsDelivr (May not be the newest version)
import SimpleCache from 'https://cdn.jsdelivr.net/gh/LmanTW/Light-Framework/Asset/Plugin/SimpleCache.mjs'
```

## Use The Plugin
```js
Light.use(SimpleCache)
```
